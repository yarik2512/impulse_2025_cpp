# Impulse 2025 C++

## Сборка проекта

```bash
chmod +x build.sh
./build.sh
```

## Запуск программы
```bash
./build/task path
```
где ```path``` - путь к входному файлу. Готовые входные файлы:
```bash
./build/task input/test_N.txt
```
где ```N``` - номер теста (```1-9```).

## Тестирование программы
```bash
./build/test_club
```
